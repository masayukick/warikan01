# 4-3. アクター定義

## 管理者（Administrator）
- **説明**: Warikan specification 2025 のシステム全体を管理する
- **前提条件**: 管理者権限でログイン済み
- **権限**: 全機能へのアクセス

## 一般ユーザー（User）
- **説明**: Warikan specification 2025 を日常的に利用する
- **前提条件**: ユーザー登録済み、ログイン済み
- **権限**: データ登録・検索・レポート出力

## 閲覧者（Viewer）
- **説明**: Warikan specification 2025 のデータを参照する
- **権限**: データ閲覧のみ
